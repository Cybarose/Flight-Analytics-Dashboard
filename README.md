# Flight Dashboard

A modern dashboard for visualizing flight data using React, Tailwind, and Recharts.  
The app can display both live API data (such as OpenSky) and offline CSV datasets containing real flight information.

---

## Features

- Displays key flight metrics, top airlines, airport performance, and flight traffic trends  
- CSV-based dataset support for offline visualization
